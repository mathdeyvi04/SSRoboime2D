# Desenvolvimento

### Fase Atual: Conexão com Servidor

Os integrantes deverão ler o seguinte [repositório](https://github.com/RCSS-IR/StarterAgent2D-V2) e compreender como funciona a conexão
de cada agente com o servidor. 

A ideia é que, conforme feito no Simulation3D, possamos iniciar o projeto com a parte mais básica possível.

### Procedimentos

##### Setup do Git

Cada aluno deverá entrar no terminal linux, em especial sua pasta home e iniciar o repositório em seus respectivos computadores
usando o comando `git clone https://github.com/mathdeyvi04/Simulation2D`.

Posteriormente, deverão obter o caminho da pasta do repositório e colocarão dentro do GitHub Desktop. Boa sorte, por isso a reunião foi aconselhada.

#### Começo do Desenvolvimento

Enfim, deverão trocar para suas respectivas branches a fim de se dedicarem às respectivas pesquisas e trabalhos.

### Dicas

- Busque por algo como `communication`, `socket`, `server`, essas palavras com certeza devem ser chave;
- **Utilize ChatGPT e os outros modelos de LLM**;
- Pergunte ao mais experiente.

# Conclusão

Na próxima reunião, ou quando completarmos essa missão, analisaremos juntos as respectivas branches, isto é, os respectivos trabalhos
e atualizaremos a `main`.
